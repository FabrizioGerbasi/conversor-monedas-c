# Conversor de Monedas en C

## Descripción
Este programa permite convertir entre quetzales, dólares y euros utilizando diferentes opciones.

## Prerrequisitos
- Compilador de C (gcc o similar)
- Terminal o consola

## Pasos para ejecutar
1. Compilar el archivo:
   gcc index.c -o conversor
2. Ejecutar el programa:
   ./conversor
3. Seleccionar la opción de conversión e ingresar la cantidad.
